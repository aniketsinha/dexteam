# Project structure

Node backend, ReactJS frontend

```
server/
  api/
    index.js
  index.js
client/
  src/
    index.js
  index.html
```

# Build 

```bash
# For local development, runs server and client mode in watch mode
npm run dev
open http://localhost:1337

# For production
npm run build

# For deployment
npm run deploy
```

We use [zeit/now](https://zeit.co/now) for deployments.
